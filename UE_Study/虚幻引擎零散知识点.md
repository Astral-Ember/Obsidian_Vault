
## Abstract关键字
在UCLASS中添加抽象标记，可以确保该类在编辑器中无法直接被实例化，类似“虚基类”
```
UCLASS(Abstract)  
class GASSTUDY_API AGAS_BaseCharacter : public ACharacter  
{  
    GENERATED_BODY()  
  
public:  
    AGAS_BaseCharacter();  
};
```
