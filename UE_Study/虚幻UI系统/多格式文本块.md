多格式文本块  
多格式文本又并富文本，主要表述了在单一文本段中出现多种文本样式（字体、大小、颜色、格式）并且还可以揉杂图像内容
 
使用多格式文本块控件的步骤：

|   |   |
|---|---|
|创建数据表格，<br><br>![Exported image](Exported%20image%2020251125141331-0.png)|选择RichTextStyleRow<br><br>![Exported image](Exported%20image%2020251125141334-1.png)|

在数据表格中点击添加，必须要有一行名为Default的数据

![Exported image](Exported%20image%2020251125141336-2.png)

添加相应格式的行，设置字体大小与颜色  
在多格式文本块中，选中对应的数据表格

![Exported image](Exported%20image%2020251125141340-3.png)

在文本中使用数据名字标签显示富文本

![Exported image](Exported%20image%2020251125141341-4.png)
