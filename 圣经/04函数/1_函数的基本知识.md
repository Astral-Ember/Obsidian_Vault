要使用C++函数，必须要：
- 提供函数定义
- 提供函数原型
- 调用函数
## 1.定义函数
函数可以分为两类：有返回值的函数和没有返回值的函数。
无返回值的函数被称为void函数，格式如下
```
void functionName (parameterList)
{
	statement (s)
	return; // optional
}
```
parameterList指定了传递给函数的参数类型和数量，return可有可无

有返回值的函数将生成一个值，并将它返回给调用函数
```
typeName functionName (parameterList)
{
	statements
	return value;// value is type cast to type typeName
}
```
对于有返回值的函数，必须使用返回语句，将值返回给调用函数
==注意==：返回结果的类型必须是typeName类型，或者可以被转换成typeName
C++对于返回值的类型有一定限制：不能是数组，但是可以是其他任何类型，如：整数、浮点数、指针、结构体和对象

## 2.函数原型和函数调用
### 1、为什么需要原型？
原型描述了函数到编译器的接口，也就是说，它将函数返回值的类型（如果有返回值的话）以及参数的类型和数量告诉编译器

### 2、原型的语法
函数原型是一条语句，因此必须以分号结束。获得原型最简单的方法是：复制函数定义中的函数头，并添加分号；然而函数原型不要求提供变量名，有类型列表足够

### 3、原型的功能
原型可以极大的降低程序出错的概率，原型确保一下几点：
- 编译器正确处理函数返回值
- 编译器检查使用的参数数目是否正确
- 编译器



函数原型是函数声明的一种形式（在主函数前声明），它为编译器提供了函数的名称、返回类型和参数列表（参数的类型和数量）。函数原型的主要作用是让编译器知道如何处理函数调用，即使函数的实际定义尚未出现在代码中。这样，函数的定义可以放在主函数main()之后，或者在其他文件中
