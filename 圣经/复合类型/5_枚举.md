C++的enum工具提供了另一种创建符号常量的方式，这种方式可以代替const。它还允许定义新类型，但必须按严格的限制进行。使用enum的句法与使用结构相似。例如，下面的语句:
`enum spectrum {red, orange, yellow, green, blue, violet, indigo, ultraviolet};`
该语句完成了两个工作：
- 让 spectrum成为新类型的名称；spectrum被称为枚举(enumeration)，就像struct变量被称为结构体
- 将red、orange、yellow等作为符号常量，它们对应整数值0~7。这些常量叫作枚举量(enumerator)
可以用枚举名来声明这种类型的变量：
`spectrum band`
在不进行强制类型转换的情况下，只能将定义枚举时使用的枚举量赋给这种枚举的变量，如下所示:
```
band = blue;// valid, blue is an enumerator
band = 2000;// invalid, 2000 not an enumerator
```

对于枚举，只定义了赋值运算符（=）。具体地说，没有为枚举定义算术运算（+、- 等）
```
band = orange;// valid
++band;// not valid, ++ discussed in Chapter 5
band = orange + red;// not valid, but a little tricky
```


枚举量是整型,可被提升为int类型，但int类型不能自动转换为枚举类型:
```
int color = blue; // valid, spectrum type promoted to int
band = 3; // invalid, int not converted to spectrum
color = 3 + red; // valid, red converted to int
```

## 1.设置枚举量的值

隐式设置：
`enum spectrum {red, orange, yellow, green, blue, violet, indigo, ultraviolet}`
默认从0开始，像数组一样

可以使用赋值运算符来显式地设置枚举量的值：
`enum bits{one = 1, two = 2, four = 4, eight = 8};`

指定的值必须是整数。也可以只显式地定义其中一些枚举量的值:
`enum bigstep{first, second = 100, third};`
first 在默认情况下是0，second是100，third是101，因为后面没有被初始化的枚举量的值比前面的枚举量大1

可以创建多个值相同的枚举量：
`enum {zero, null = 0, one, numero_uno = 1};`
