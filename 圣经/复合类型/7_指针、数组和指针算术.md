指针和数组基本等价的原因在于指针算术(pointer arithmetic)和C++内部处理数组的方式。
C++将数组名解释为地址
```
	int* p;
	int array[5] = { 1,2,3,4,5 };
	p = array;
	cout << p << ' ' << &array[0] << ' ' << '\n';
	cout << *p << ' ' << array[0] << '\n';
	cout << p + 1 << ' ' << &array[1] << '\n';
	cout << *(p + 1) << ' ' << array[1] << '\n';
```
p记录的是array第一个元素的地址
`*p`记录的是第一个元素的值
如果p+1，由于存储的是int类型，那么地址将增加8，p为第二个元素的地址，相应的`*p`就是第二个元素的值


### 1.使用new创建动态结构体
在运行时创建数组优于在编译时创建数组，对于结构也是如此。需要在程序运行时为结构分配所需的空间，这也可以使用new运算符来完成。通过使用new，可以创建动态结构。同样，“动态”意味着内存是在运行时，而不是编译时分配的。由于类与结构非常相似，因此本节介绍的有关结构的技术适用于类

`结构体名* 指针名 = new 结构体名`

```
struct Student
{
	string name;
	int age;
};
int main()
{
	Student std = { "qry",21 };
	Student* p = &std;
	Student* p2 = new Student;
	p2 = &std;
	cout << p->name << '\n';
	cout << p2->age;
}
```
结构体指针使用，箭头运算符（->）读取数据
