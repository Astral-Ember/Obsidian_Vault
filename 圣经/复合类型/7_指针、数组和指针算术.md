指针和数组基本等价的原因在于指针算术(pointer arithmetic)和C++内部处理数组的方式。
C++将数组名解释为地址
```
	int* p;
	int array[5] = { 1,2,3,4,5 };
	p = array;
	cout << p << ' ' << &array[0] << ' ' << '\n';
	cout << *p << ' ' << array[0] << '\n';
	cout << p + 1 << ' ' << &array[1] << '\n';
	cout << *(p + 1) << ' ' << array[1] << '\n';
```
p记录的是array第一个元素的地址
`*p`记录的是第一个元素的值
如果p+1，由于存储的是int类型，那么地址将增加8，p为第二个元素的地址，相应的`*p`就是第二个元素的值
