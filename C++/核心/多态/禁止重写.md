2. 禁止重写

如果我们不希望一个类可以被继承  
我们可以通过 final 关键词来修饰类  
基本语法：  
class 类名 final  
{  
}  
在类名后加上 final 关键字，就可以禁止该类被继承了
 
final 关键字除了可以禁止类被继承  
还可以限定函数不能再被重写  
语法规则：  
在函数尾部加上 final  
它只能搭配 virtual 和 override 使用  
virtual 返回类型 函数名（参数列表） final;  
返回类型 函数名（参数列表） override final;  
因为它是用来禁止虚函数被重写的  
同时可以让子类也无法去写一个同名函数
