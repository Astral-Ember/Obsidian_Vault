其实 string 的本质就是一个 C++ 中封装的一个管理字符数组的类  
它封装了动态内存分配、长度管理、字符串操作函数等复杂逻辑  
因此在现代 C++ 开发当中，都建议大家使用 string 来替代字符数组  
因为这样更高效、安全
