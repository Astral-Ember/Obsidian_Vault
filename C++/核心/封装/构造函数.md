2. 委托构造函数

构造函数是一个特殊的成员函数  
是用来对类对象进行初始化用的  
当我们声明一个类变量（对象）时，会自动调用构造函数
 
2-4. 两种赋值方式  
2-4-1: 在函数中赋值  
2-4-2: 利用成员初始化列表来初始化（更高效）  
2-5. 若传入参数和成员变量重名，可以使用 this-\> 成员变量 来表示是成员变量  
注意：如果声明成员变量时也初始化了 那么以构造函数中初始化的值为准  
3-1. 栈上分配 - 不带 new  
3-1-1. 类型名 类对象(参数...)  
3-1-2. 类型名 类对象{参数…}  
3-1-3. 类型名 类对象 = 类型(参数...);  
3-1-4. 类型名 类对象 = 类型{参数…};  
3-1-5. 类型名 类对象 = 类型{参数…};  
3-2.堆上分配-带new  
类型* 类对象 = new 类型(参数);
 
所谓的的委托构造函数  
就是允许一个构造函数调用另一个构造函数，以减少代码重复  
基本语法：  
构造函数(): 另一个重载的构造函数(参数)  
{}

|   |
|---|
|Function::Function(int x):x(x)  <br>{  <br>y = 0;  <br>z = 0;  <br>}  <br>Function::Function(int a, int b):Function(a)  <br>{  <br>this-\>y = b;  <br>z = 0;  <br>}  <br>//先调用上面的构造函数给x赋值|

主要作用：  
在调用某一个构造函数之前可以通过这种方式先调用另一个构造函数  
达到复用相同代码的作用，这样我们在一些构造函数中就可以少写一些代码逻辑了
