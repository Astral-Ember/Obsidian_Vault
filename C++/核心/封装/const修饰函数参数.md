const修饰函数参数

1. 基本类型
2. 指针
3. 引用

规则：返回值 函数名（const int等基础类型 参数名）  
作用：不允许传入的参数在内部修改（但是意义不大，就算内部修改也不会影响外部）  
指向能改，值不能改（相当于指向常量的指针）  
规则：返回值 函数名（const 类型* 参数名）  
作用：该指针可以修改指向，但是指向的值不能修改，主要用于放置修改指针指向内容的值  
指向不改变，值能改（相当于指针常量）  
规则：返回值 函数名（ 类型* const 参数名）  
作用：该指针不可以修改指向，但是指向的值能修改，主要用于放置修改指针指向内容的值  
指向不能改，值也不能改（相当于指向常量的指针常量）  
规则：返回值 函数名（ const 类型* const 参数名）  
作用：该指针可以修改指向，但是指向的值不能修改，主要用于放置修改指针指向内容的值  
一般推荐使用第一种方式，可以提高代码安全性  
规则：返回值 函数名（const 类型& 参数名）  
作用：常量对象和非常量对象都可以传入，但是 不能修改对象本身 ，由于是引用 ，还可以避免拷贝开销
