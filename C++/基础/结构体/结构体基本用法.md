结构体是一种自定义变量类型 类似枚举需要自己定义  
它是数据和函数的集合  
在结构体中可以声明各种变量和方法  
作用：  
用来表现 存在关系的数据集合 比如用结构体表现 学生 动物 人类 等等  
结构体关键字 struct

|   |
|---|
|struct 自定义结构体名字  <br>{  <br>//第一部分：变量  <br>//第二部分：构造函数和析构函数（可选）  <br>//第三部分：成员函数（可选）  <br>};  <br>//e.g.  <br>struct Student  <br>{  <br>string name;  <br>int age;  <br>float score;  <br>Student* s;//可以声明自己类型的结构体变量 但一定是指针或引用  <br>Student& s2;  <br>//函数（方法）表示它的行为，并可以直接操作它的成员变量  <br>void speak()  <br>{  <br>cout \<\< "my name is:" \<\< name \<\< ", age is: " \<\< age \<\< ", score is: " \<\< score \<\< endl;  <br>}  <br>};|

变量类型可以写任意的类型 包括 结构体（但是不能是自己的结构体变量 指针和引用除外）  
在声明结构体对象后直接通过{}按变量顺序进行初始化  
聚合初始化允许通过大括号 {} 的方式直接对聚合类型的成员变量逐个赋值

|   |
|---|
|struct Student  <br>{  <br>string name;  <br>int age;  <br>float score;  <br>Student* s;  <br>//Student& s2;  <br>};  <br>Student s1 = { "张三",20,99.5,nullptr };|

不可以直接跳过某一些成员进行初始化，但是可以不初始化或只初始化前几个  
定义结构体类型的同时声明结构体变量  
直接在结构体类型声明的大括号后声明变量

|   |
|---|
|struct Student  <br>{  <br>string name;  <br>int age;  <br>float score;  <br>Student* s;  <br>//Student& s2;  <br>}ss1, ss2 = { "张三",20,99.5,nullptr };|

省略结构体类型名 直接声明结构体类型变量

|   |
|---|
|struct {  <br>int test1;  <br>string test2;  <br>}test1 = {1,"123"};|

只能用直接声明方式  
可以在堆上分配结构体对象 一定要记得销毁
