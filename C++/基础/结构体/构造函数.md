基本概念  
构造函数是一个特殊的成员函数  
是用来对结构体对象进行初始化用的  
基本规则  
声明在结构体中

1. 没有返回值
2. 函数名必须和结构体名相同
3. 可以被重载
4. 两种赋值方式
5. 若传入参数和成员变量重名，可以使用 this-\> 成员变量 来表示是成员变量

在函数中赋值  
利用成员初始化列表来初始化（更高效）

|   |
|---|
|int age;  <br>string name;  <br>Student()  <br>{  <br>age = 18;  <br>name = "zhangsan";  <br>}  <br>Student(int age, string n):age(age),name(n)  <br>{  <br>}  <br>Student(int age,string n)  <br>{  <br>this-\>age = age;  <br>name = n;  <br>}|

调用构造函数的几种方式  
栈上分配 - 不带 new  
结构体类型 结构体变量（参数...）  
结构体类型 结构体变量 {参数...}  
结构体类型 结构体变量 = 结构体类型（参数...）；  
结构体类型 结构体变量 = 结构体类型{参数...}；  
结构体类型 结构体变量 = （参数...）；  
结构体类型 结构体变量 = {参数...}；

|   |
|---|
|Student s1(15,"hhh");  <br>cout \<\< s1.age \<\< ' ' \<\< s1.name \<\< '\n';// 15 hhh  <br>Student s2{ 20 };  <br>cout \<\< s2.age \<\< '\n';// 20  <br>Student s3 = Student();  <br>cout \<\< s3.age \<\< ' ' \<\< s3.name \<\< '\n';// 18 zhangsan|

堆上分配 - 带 new  
结构体类型 结构体变量 = new 结构体类型(参数);

|   |
|---|
|cout \<\< s4-\>age \<\< ' ' \<\< s4-\>name \<\< '\n';// 18 zhangsan  <br>delete s4;  <br>s4 = nullptr;|

结构体还有两个默认的构造函数  
传入 const 左值引引用的构造  
主要作用：避免不必要的复制，允许使用已有的对象用于初始化成员  
一般用来处理复制另一一个对象的数据  
传入 右值引引用的构造  
主要作用：接收将要被移动的临时对象或其他右值，可以提高性能
 
如果我们没有为结构体定义任何构造函数  
编译器会生成一个默认的构造函数  
如果定义了其他构造函数（如带参数的构造函数）  
则默认构造函数不会被自动生成  
除非我们显式定义它  
注意：默认构造函数就是一个无参的构造函数  
如果声明了其他有参构造的情况下 之后希望使用无参构造函数去声明对象 需要显式声明一个无参构造函数
