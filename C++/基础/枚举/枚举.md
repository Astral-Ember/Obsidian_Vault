枚举是一个比较特别的存在  
它是一个被命名的整型常量的集合  
是一种用户自定义的数据类型  
一般用它来表示状态 类型等  
注意：声明枚举和声明枚举变量是两个概念  
声明枚举：相当于创建一个自定义枚举类型  
声明枚举变量：使用变量的自定义枚举类型 创建一个枚举变量  
enum 枚举类型{枚举项名字，枚举项名字1…}

|   |
|---|
|enum E_枚举名字  <br>{  <br>枚举项名字,//0 如果不自己赋值 默认第一项为0 之后枚举项对应的值会一次累加  <br>枚举项名字2,//1  <br>枚举项名字3,//3  <br>//...n个枚举项都可以 根据需求添加  <br>};  <br>enum E_EnumName  <br>{  <br>EnumItemName,//0 如果不自己赋值 默认第一项为0 之后枚举项对应的值会一次累加  <br>EnumItemName2 = 5,//5  <br>EnumItemName3,//6  <br>//...n个枚举项都可以 根据需求添加  <br>};  <br>//枚举变量声明规则：  <br>//枚举类型 变量名 = 枚举中包含的枚举项名字  <br>E_枚举名字 变量名1 = 枚举项名字2;  <br>E_EnumName variableName1 = EnumItemName3;  <br>cout \<\< 变量名1 \<\< endl;  <br>cout \<\< variableName1 \<\< endl;|

中文不会报错但是最好使用英文  
枚举变量声明规则：  
枚举类型 变量名 = 枚举中包含的枚举项名字
 
枚举的声明一般都是作为全局变量进行声明  
声明不写在任何语句块中  
枚举变量的声明可以写在任意位置  
注意： 声明枚举变量时要确保枚举在之前进行了声明
 
|   |
|---|
|enum Color { RED, GREEN, BLUE };  <br>int main()  <br>{  <br>Color color = RED;  <br>//枚举的第一个作用：提高代码可读性  <br>if(color==RED)  <br>{  <br>cout \<\< "The color is red." \<\< endl;  <br>}  <br>else if(color==GREEN)  <br>{  <br>cout \<\< "The color is green." \<\< endl;  <br>}  <br>else if(color==BLUE)  <br>{  <br>cout \<\< "The color is blue." \<\< endl;  <br>}<br><br>  <br><br>//第二个作用： 枚举和switch语句结合使用  <br>switch(color)  <br>{  <br>case RED:  <br>cout \<\< "The color is red." \<\< endl;  <br>break;  <br>case GREEN:  <br>cout \<\< "The color is green." \<\< endl;  <br>break;  <br>case BLUE:  <br>cout \<\< "The color is blue." \<\< endl;  <br>break;  <br>default:  <br>cout \<\< "Unknown color." \<\< endl;  <br>break;  <br>}|

枚举转整型可以隐式转换  
整型转枚举  
需要使用 static_cast\<枚举类型\>(整型值）方法强制转换

|   |
|---|
|cout \<\< color \<\< '\n';//0  <br>color = static_cast\<Color\>(1);  <br>cout \<\< color \<\< '\n';//1  <br>switch(color)  <br>{  <br>case RED:  <br>cout \<\< "The color is red." \<\< endl;  <br>break;  <br>case GREEN:  <br>cout \<\< "The color is green." \<\< endl;//因为color被赋值为1，所以会执行这里  <br>break;  <br>case BLUE:  <br>cout \<\< "The color is blue." \<\< endl;  <br>break;  <br>default:  <br>cout \<\< "Unknown color." \<\< endl;  <br>break;  <br>}|

在游戏开发中 对象很多时候有很多状态  
比如玩家有一个动作状态 我们需要用一个变量或者标识来标识当前玩家处于哪一种状态（状态机）  
综合考虑 可以使用int来标识它的状态  
1 行走  
2 待机  
3 奔跑  
…  
这个时候就可以通过枚举来帮助我们分清楚状态的含义，以及进行逻辑处理
