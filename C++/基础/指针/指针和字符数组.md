1. 字符数组就是类型为字符 char 的一维数组  
2. 字符数组可以用字符串初始化（默认会在结尾加上 \0 结束符号）  
char charArr[] = "Hello World";  
3. 字符数组可以直接用于打印，但是需要用 \0 表示结束  
char charArr2[] = { 'A', 'B', 'C' };  
会报错（越界），直到遇到 \0 才会结束，会导致出现乱码  
cout \<\< charArr2 \<\< endl;  
遇到 \0 就会停止  
char charArr3[] = { 'A', 'B', 'C', '\0' };  
cout \<\< charArr3 \<\< endl;  
指向常量的指针：const 在 * 前面去添加即可，值不能改，指向可以改  
指针常量：const 在 * 后面去添加即可，值可以改，指向不能改  
指向常量的指针常量：const 在 * 前后都加，值和指向都不能改了  
char* ptrChar = charArr;  
ptrChar = &charArr[0];
