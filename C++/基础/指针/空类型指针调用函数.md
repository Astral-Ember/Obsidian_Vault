空类型指针  
空类型指针是一种特殊的指针  
它表示不特定类型的指针  
指针变量可以指向任意的变量类型  
具体存放何种类型，由后续赋值决定  
赋值后，如果需要使用需要强转为对应的数据类型

|   |
|---|
|void* p = nullptr;  <br>int a = 10;  <br>p = &a;  <br>cout \<\< *(int*)p \<\< endl;|

空类型指针可以指向任意类型的函数

|   |
|---|
|void test()  <br>{  <br>}  <br>int add(int a, int b)  <br>{  <br>return a + b;  <br>}  <br>int main()  <br>{  <br>void* q = nullptr;  <br>q = test;  <br>q = add;  <br>}|

必须将空类型指针强转为 返回值类型、参数类型 符合规范的指针后使用  
注意：  
强转时省略函数指针名即可

|   |
|---|
|void test()  <br>{  <br>cout \<\< "test function" \<\< endl;  <br>}<br><br>  <br><br>int add(int a, int b)  <br>{  <br>return a + b;  <br>}  <br>int main()  <br>{  <br>void* q = test;  <br>((void(*)())q)();  <br>q = test;  <br>q = add;  <br>cout\<\<((int(*)(int, int))q)(2, 3);  <br>}|

((返回值类型(*)(参数类型))空类型指针名 )(传入参数)
