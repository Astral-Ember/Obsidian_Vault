数组在内存中是连续存储的  
因此我们只要利用指针获取到数组一开始的地址就可以通过指针的自增减获取到数组中所有的信息  
int array[5]={1,2,3,4,5};  
int* i = nullptr;  
int* j = &array[0];  
数组变量名即可表示数组的首地址  
i=array;  
cout\<\<i\<\<'\n';  
cout\<\<j\<\<'\n';  
首地址便是第一个元素

|                                                                                            |
| ------------------------------------------------------------------------------------------ |
| int array[5]={1,2,3,4,5};  <br>int* i = nullptr;  <br>i=array;  <br>cout\<\<*i\<\<'\n';//1 |

还可以用循环

|   |
|---|
|int array[5] = { 4,2,5,1,7 };  <br>int* p = nullptr;  <br>p = array;  <br>for (int i = 0; i \< 5; i++)  <br>{  <br>cout \<\< *(p + i) \<\< '\n';  <br>}|

一定注意：不能越界  
数组名不能像刚才指针一样 去改变它指向的内存地址  
我们只能利用它的地址来进行计算 不能改变（地址）

|                                                                                                              |
| ------------------------------------------------------------------------------------------------------------ |
| int array[5] = { 4,2,5,1,7 };  <br>int* p = nullptr;  <br>p = array;  <br>cout \<\< *(array+1) \<\< '\n';//2 |

注意：我们无法直接通过指针获取到数组的容量  
因为指针其实只是记录了一个地址而已，通过地址并不能知道数组容量

|   |
|---|
|int array[5] = { 4,2,5,1,7 };  <br>int* p = nullptr;  <br>int* q = array;  <br>p = array;  <br>for (int i = 0; i \< sizeof(array)/sizeof(int); i++)  <br>{  <br>//通过偏移获取数组元素  <br>cout \<\< *(p + i) \<\< '\n';  <br>//通过指针自增获取数组元素  <br>cout \<\< *q++ \<\< '\n';  <br>//通过数组名获取数组元素 但是不能改变数组名的指向  <br>cout \<\< *(array + i) \<\< '\n';  <br>}|

指针变量类型需要和数值保持一致  
数组名字和数组首元素的地址 都可以获取到数组开始的位置（首地址）  
利用指针的自增自减（加减运算）可以获取到数组中不同的元素，需要注意越界问题  
数组名可以用来当作指针使用，但是我们不能改变数组指向的内容（不能对数组进行重指向）
