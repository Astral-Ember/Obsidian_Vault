多级指针就是指向指针的指针  
它可以理解为一种指针的嵌套结构  
是用来存储指针变量的地址的  
就是最普遍的指针  
二级指针是用来指向一级指针变量地址的指针  
e.g.

|   |
|---|
|int a= 10;  <br>int* p = &a;  <br>int** pp = &p;<br><br>  <br><br>cout \<\< &p \<\< '\n';//输出p的地址  <br>cout \<\< &a \<\< '\n';  <br>cout \<\< p \<\< '\n';//存储的值是a的地址：&a；与上面一行输出的值相同  <br>cout \<\< pp \<\< '\n';//存储的值是p的地址：&p；与第一行输出的值相同  <br>cout \<\< *pp \<\< endl;//对pp解引用，输出p的值，即a的地址：&a  <br>cout \<\< **pp \<\< '\n';//对pp解两次引用，输出a的值：10|

多级指针的本质就是用来存储 指针变量的地址 的  
多级指针可以有几级，第 n 级 指针是用来存储 n-1 级指针的地址的  
如果想要获取多级指针最底层的值，只需要再前面加 n 个 * 即可
