左值引用和右值引用

1. 左值和右值的含义
2. 左值引用
3. 右值引用

左值和右值是表达式结果的分类 代表表达式再计算时的特性 主要用于区分持久性和是否具有存储位置  
左值：  
表示在表达式结束后仍然存在的对象，通常可以被赋值的，一般在一定周期内的永久存在的  
特点：  
左值有明确的存储位置（可以获取到地址）  
可以出现在赋值符号的左边，允许被赋值修改  
e.g.

|   |
|---|
|int x = 5;//x就是左值，有存储位置  <br>x = 10;//可以被赋值  <br>int* p = &x;|

右值：  
表示临时对象或不具持久性的值，一般在表达式计算完成后即被释放  
特点：  
右值没有明确的存储位置（一般不能获取地址）  
右值表达式一般只能出现在赋值符号的右边  
右值一般是临时计算的结果，或者没有名称的值，例如字面值、临时对象等

上一节的[引用的基础使用](引用的基础使用.md)中使用的引用就是左值引用使用  
引用符号 & 来进行引用变量的声明  
左值引用只能绑定到左值（即在表达式结束后仍存在的变量）  
主要用于指向持续存储在内存中的变量或对象  
没有办法指向右值  
右值引用使用是使用 两个 && 声明的引用对象  
它只能绑定到右值（即临时的、没有名称的对象，比如函数返回值、字面量等）  
特点：  
可以指向其右值  
右值引用延长了右值的生命周期  
注意：  
我们目前可能不太感受到右值引⽤的意义  
随着我们的学习，以后我们会渐渐明白的
