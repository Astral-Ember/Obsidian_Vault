引用和函数

1. 使用引用参数传递函数
    1. 左值引用参数
    2. 右值引用参数
2. 常量引用参数
3. 函数返回引用

用于将函数形参绑定到修改的左值变量，避免值拷贝并且允许函数内修改传入的对象  
它的作用类似使用指针传参变量参数，但是是形参不会重新创建副本，而是直接和实参绑定，是实参的别名  
但是还是必须遵守引用使用的规则：

1. 引用的类型必须和引用的内容类型一致
2. 引用绑定的对象不能改动
3. 引用不会占用新的内存空间，只是原变量的另一种名称而已
4. 通过引用传递的参数，当函数执行完毕后并不会释放该对象，具体何时释放取决于外部的情况

e.g.

|   |
|---|
|void func(int &x)  <br>{  <br>x = 10;  <br>}  <br>int main()  <br>{  <br>int a = 20;  <br>func(a);  <br>cout \<\< a;//10  <br>}|

右值引用的主要用途是优化性能  
避免临时对象或大对象的拷贝，可以直接拿走这些对象的资源  
注意：  
我们目前可能不太感受到右值引用的意义  
随着我们的学习，以后我们会渐渐明白的  
e.g.

|   |
|---|
|void func(int &&x)  <br>{  <br>cout \<\< x \<\< '\n';  <br>}  <br>int main()  <br>{  <br>int a = 20;  <br>func(20);//20  <br>}|

对于右值引用参数 我们可以利用它少分配一些中间的临时空间  
就可以达到优化性能，节约内存的使用，直接把想获取的内容当作右值传入函数即可  
可以在函数中声明引用参数时，在前面加上const关键字 让其变成一个常量引用参数  
通常用作只读操作，确保函数内只用不能改
