数组是存储一组相同类型数据的集合  
数组分为 一维、 多维  
一般情况 —— 一维数组 就简称为 数组  
数组的存储序列是线性的  
意思就是数组中的数据之间 在内存中是紧挨着的  
声明一个指定容量的数组，但每个房间的值不确定  
变量类型 可以是。。随便一个

|   |
|---|
|int array[6];|

通过这种方式声明一个数组 相当于开了6个小房间 但是房间里面的int值不确定  
声明一个指定容量的数组，每个房间的值在声明时进行初始化

|   |
|---|
|int array[6] = { 1,2,3,4,5,6 };|

声明一个不指定容量的数组，具体容量由编译器根据初始值的数量自动推导数组大小

|   |
|---|
|int array[] = { 1,2,3,4 };|

利用 sizeof 进行计算

|   |
|---|
|int array[ ] = { 1,2,3,4 };  <br>cout \<\< sizeof(array) / sizeof(array[0]) \<\< '\n';//4  <br>//sizeof(array)计算整个数组的大小，sizeof(array[0])计算数组中每个元素的大小|

数组中的下标和索引 他们是从 0 开始的  
通过 索引下标去 获得数组中某一个元素的值时  
注意

我们目前学习的数组相关知识，属于静态数组相关知识  
声明一个静态数组时， 数组的大小是固定的，并且必须在编译时确定  
声明后， 不能修改数组的大小或将其指向另一个数组  
静态数组的大小和类型在声明时固定，不能在运行时修改  
与之对应的动态数组在学习了堆内存分配后再讲解  
概念：同一变量类型连续存储的数据集合  
一定要掌握的数集相关知识：声明、遍历、增删查改  
所有的变量类型都可以声明为数组  
它是用于批量存储游戏中同一类型的对象的数据结构
