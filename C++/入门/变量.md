变量  
变量声明：变量类型 变量名 = 初始值;

1. ==有符号的整形变量==
    1. 有符号断整形
    
    short([signed] short [int])  
    存储范围：-32768~32767
    
    3. 有符号整型
    
    int ([signed] int)  
    存储范围：-21亿多~21亿多
    
    5. 有符号长整型
    
    long([singed] long [int])  
    存储范围：-21亿多~21亿多 或 -9223亿多~ 9223亿亿多  
    不同的原因是因为操作系统不同
    
    7. 有符号长长整型
    
    Long long ([singed] long long [int])  
    存储范围：-9223亿多~ 9223亿亿多
    
      
    
2. ==无符号的整型变量==
    1. 有符号的短整型
    
    Unsigned short ([int])  
    存储范围：0~65535
    
    3. 无符号整型：
    
    Unsigned int  
    存储范围：0~42亿多
    
    5. 无符号长整型
    
    Unsigned long  
    存储范围：0~42亿多或0~18466亿亿多
    
    7. 无符号长长整型
    
    Unsigned longlong  
    存储范围：0~42亿多或0~18466亿亿多
    
3. ==浮点数（小数）==
    1. 单精度浮点数
    
    Float  
    存储范围：-3.4*10^-38~3.4*10^-38  
    关键点：  
    1.声明时在小数数值后面加一个f，因为在C++中声明的小数默认是double类型，加f是告诉系统是float类型  
    2.存储6/7位有效数字（对于过大和过小的数，并不能保证存储数值的准确性）  
    当超过有效数字后，会对后面的数进行四舍五入，用cout打印浮点数，会默认转换为6位有效数字的精度输出
    
    3. 双精度浮点数
    
    Double  
    存储范围：-1.7*10^-308~1.7*10^-308  
    关键点：  
    1.C++中默认声明小数是double类型  
    2.存储15/16位有效数字
    
    5. 长双精度浮点数
    
    Long double  
    存储范围：-1.1*10^4932~1.1*10^4923  
    （不同操作系统存储空间不同）  
    关键点：  
    1.需要加l/L后缀  
    2.存储18/21、33/36位有效数字
    
4. ==特殊类型==
    1. 布尔类型
    
    Bool true false 表示真假的数据类型  
    true可以代表1 false表示0
    
    3. 字符类型
    
    Char 用来存储单个字符的变量类型  
    但是不能用来存储所有字符，只能来存储ASCII编码的字符  
    C++中 char也能当作整型对待 字符对应的整数就是它的ASCII码值
    
    5. 字符串类型
    
    String 用来存储多个字符 没有上限，并且它可以用来存储任何字符，因为它采用的编码规则更加广泛
    
5. ==为什么有那么多不同的变量类型==
    1. 不同的变量 存储的范围和类型不一样 本质是占用的内存空间不同 选择不同的数据类型装载不同的数据
6. ==多个相同类型变量同时声明==
    1. 变量类型 变量名 = 初始值， 变量名 = 初始值 …;
    
    |   |
    |---|
    |int a = 0, b = 1, c = 2;|
    
    也可以不初始化
