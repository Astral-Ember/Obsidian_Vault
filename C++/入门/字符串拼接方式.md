之前的算数运算符 是数值类型变量进行数学运算的  
而 string 不存在使用算数运算符进行数学运算  
但是可以通过 + 号来进行字符串拼接  
注意：不能和数值类型直接进行拼接但是可以和字符char进行拼接  
用 + 号进行字符串拼接  
复合运算符 += 进行 字符串拼接  
注意： 用 + 号拼接 是用运算符拼接的唯一方法 不能用 -*/%...  
e.g.

|   |
|---|
|string str;  <br>str="1";  <br>str += "C";  <br>char ch = 'A';  <br>str = str + ch + "B" + "123";  <br>//下面这种形式要尽量避免  <br>//原因：先算右侧再和左侧进行计算 右侧是两个char相加 本质会转换成int相加  <br>//得到的结果可能对应的字符不存在，如果存在也可能不是想要的结果  <br>str += ch + "c";|
