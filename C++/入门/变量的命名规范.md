变量的命名规范

1. ==必须遵守的规则==
2. ==常用的命名规范==
    1. 驼峰命名法—首字母小写，之后单词首字母大写（变量、函数） — C++中常用
    2. 帕斯卡命名法—所有单词首字母都大写（类）
    3. 下划线命名法—单词之间用下划线隔开（变量、函数） — C++中常用
    4. 匈牙利命名法—在变量前加上一个表示类型的前缀或表示用途的前缀（变量）
    
    变量类型前缀常用规则：  
    int → n  
    float → f  
    char → c  
    bool → b  
    string → str  
    用途类型前缀常用规则：  
    全局变量 → g_  
    静态变量 → s_  
    类成员变量 → m_
    
    6. 前缀下划线命名法—在私有变量前加上 _（私有变量）
    7. C++中 对大小写是敏感的 是有区分的

不能重命名(同一语句块中)  
不能以数字开头  
不能使用程序关键字命名  
不能有特殊符号（下划线等符号除外）  
建议的命名规则：变量名要有含义 → 用英文（拼音）表示变量的作用  
非常不建议的命名规则：用汉字命名
