异常捕获

2. 异常捕获的基本语法
3. 实践

异常捕获可以避免当代码报错时 造成程序卡死 无法继续往下执行  
将玩家输入的内容存储在 string类型的变量（容器）中

|   |
|---|
|string str;  <br>cin \>\> str;  <br>将字符串转为数值类型时必须合法合规，因此会报错  <br>int i = stoi(str);  <br>cout \<\< i \<\< '\n';|

|   |
|---|
|try  <br>{  <br>// 希望进行异常捕获的代码块  <br>// 放到try中  <br>// 如果try中的代码报错了不会让程序卡死  <br>}  <br>catch (const exception&)  <br>{  <br>//如果出错了 会执行 catch中的代码 来捕获异常  <br>//catch(const std::exception&) 具体报错跟踪 可以利用传入的对象获取到相应报错内容  <br>}  <br>//注意：异常捕获代码结构中不需要加分号，在里面写代码逻辑时 每一句才加分号|

|   |
|---|
|try  <br>{  <br>string str;  <br>cin \>\> str;  <br>int i = stoi(str); //可能会出错的代码 放在 try中执行  <br>cout \<\< "转换成功，输入的数字是：" \<\< i \<\< endl;  <br>}  <br>catch (const exception&)//如果报错才会执行 catch 代码块  <br>{  <br>cout \<\< "转换失败，请确保输入的是一个合法的数字字符串" \<\< '\n';  <br>}|
