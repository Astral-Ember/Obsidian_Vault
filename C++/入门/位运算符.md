位运算符主要用数值类型进行计算  
将数值转换为2进制 再进行位运算  
链接两个数值进行位计算 将数值转为2进制  
对位运算 有0则0

|   |
|---|
|int a = 1;//001  <br>int b = 5;//101  <br>//001  <br>//& 101  <br>//001=1  <br>int result = a & b;  <br>cout \<\< result;//result=1|

没有括号就从左到右依次计算  
连接两个数值进行位计算 将数值转为2进制  
对位运算 有1则1  
多个数值连续计算和之前的规则相同

|   |
|---|
|int a = 1;//001  <br>int b = 5;//101  <br>//001  <br>//\| 101  <br>//101  <br>int result = a \| b;  <br>cout \<\< result;//result=5|
 
连接两个数值进行位计算 将数值转为2进制  
对位运算 相同为0 不同为1

|   |
|---|
|int a = 1;//001  <br>int b = 5;//101  <br>//001  <br>//^ 101  <br>//100  <br>int result = a ^ b;  <br>cout \<\< result;//result=4|

写在数值前面，将数值转换为2进制  
对位运算 0变1 1变0

|   |
|---|
|short a = 5;//0000 0000 0000 0101  <br>// 补码 反码 原码  <br>cout \<\< ~a;//1111 1111 1111 1010 -1 = 1111 1111 1111 1001 = 1000 0000 0000 0110 = -6|

让一个数的2进制数进行左移和右移  
左移几位 右侧加几个0

|   |
|---|
|short a = 5;//101  <br>short result = a \<\< 5;//10100000  <br>cout \<\< result \<\< endl;//result is 160|

右移几位 右侧去掉几个数

|   |
|---|
|short a = 5;//101  <br>short result = a \>\> 2;//101 \>\> 2 = 001  <br>cout \<\< result \<\< endl;//result is 1|
